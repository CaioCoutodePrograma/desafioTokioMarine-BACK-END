# Desafio Tokio Marine BACKEND


### Linguagem: 

- Java 11
### FrameWorks: 

-  Spring
-  Hibernate
-  JPA

### Gerenciador de Dependências:

- Apache Maven 3.8.8

### IDE:

- IntelJ

### Data Base: 

- H2

Para Subir o projeto basta instalar Java 11 , Maven e IntelJ e baixar a dependências.
Iniciar o projeto pela classe DesafioApplication.java
